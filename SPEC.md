## 1. الهوية البصرية (Brand Identity)

### الاسم: **دَرْب**
- Tagline: **"كل واحد عنده درب"**

### مبدأ التصميم الأساسي
> **"أبسط ما يمكن" - التطبيق لازم تفهمه الجدة اللي ما استخدمت جوال قبل.**
- أيقونات كبيرة مع نصوص
- ألوان واضحة ومتباينة
- خطوات قليلة لكل عملية (أقل من 3 نقرات للوصول للهدف)
- دعم للصوت مستقبلاً (اللي ما يعرف يقرأ)
- أزرار واضحة (OK / Cancel) بدون خيارات معقدة

### ملاحظات الفئات
- كل فئة رئيسية لها **أيقونة كبيرة + لون مميز**
- الفئات الفرعية تكون **اختيارات باللمس** (قوائم منسدلة)
- المستخدم **يختار فقط** ولا يكتب أي فئة
- في حال الخدمة ما تنطبق تحت أي فئة، يختار "خدمات أخرى"
- إضافة فئة جديدة باقتراح من الأدمن (موافقة يدوية)

## 4. نسبة المنصة

| الفترة | النسبة |
|---|---|
| أول 6 شهور | **0% (مجاني)** |
| بعد 6 شهور | **15%** من قيمة الصفقة (قابلة للتعديل) |

> نسبة 15% تشمل تكاليف التشغيل، الدفع الإلكتروني، فريق النزاعات، والتسويق.

## 5. قاعدة البيانات (Database Schema)

### Users (المستخدمين)

```sql
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    phone VARCHAR(15) UNIQUE NOT NULL,
    name VARCHAR(100) NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    role VARCHAR(20) DEFAULT 'user', -- user | admin
    city VARCHAR(100),
    avatar_url TEXT,
    id_verified BOOLEAN DEFAULT FALSE,
    id_image_url TEXT, -- صورة البطاقة
    status VARCHAR(20) DEFAULT 'active', -- active | suspended | banned
    rating DECIMAL(2,1) DEFAULT 0,
    completed_orders INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE user_devices (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id) ON DELETE CASCADE,
```

```sql
CREATE TABLE services (

    views INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
```

### Transactions (الصفقات)

```sql
CREATE TABLE transactions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    request_id UUID REFERENCES requests(id),
    offer_id UUID REFERENCES offers(id),
    service_id UUID REFERENCES services(id),
    client_id UUID REFERENCES users(id) NOT NULL,
    provider_id UUID REFERENCES users(id) NOT NULL,
    amount DECIMAL(10,2) NOT NULL,
    platform_fee DECIMAL(10,2),
    total_amount DECIMAL(10,2) NOT NULL,
    payment_type VARCHAR(20) DEFAULT 'full', -- full | milestone

    client_confirmed BOOLEAN DEFAULT FALSE,
    dispute_winner VARCHAR(10), -- client | provider
    completed_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Wallets (المحافظ)

```sql
CREATE TABLE wallets (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id) ON DELETE CASCADE UNIQUE,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

CREATE TABLE wallet_transactions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

);
```

```sql
CREATE TABLE withdrawals (

    wallet_type VARCHAR(30) NOT NULL, -- karimi | jeeb | onecash | joali | bank

    notes TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Reviews (التقييمات)

```sql
CREATE TABLE reviews (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    transaction_id UUID REFERENCES transactions(id) ON DELETE CASCADE,
    reviewer_id UUID REFERENCES users(id) ON DELETE CASCADE,
    reviewee_id UUID REFERENCES users(id) ON DELETE CASCADE,
    rating INT NOT NULL CHECK (rating >= 1 AND rating <= 5),
    comment TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(transaction_id, reviewer_id)
);
```

### Notifications (الإشعارات)

```sql
CREATE TABLE notifications (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

    title VARCHAR(200) NOT NULL,
    body TEXT,
    data JSONB, -- بيانات إضافية (transaction_id, request_id, etc)

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Disputes (النزاعات)

```sql
CREATE TABLE disputes (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

    reason TEXT NOT NULL,
    evidence JSONB DEFAULT '[]', -- صور/ملفات
    status VARCHAR(20) DEFAULT 'open', -- open | under_review | resolved
    resolution VARCHAR(30), -- full_to_provider | full_to_client | partial | cancelled
    resolved_by UUID REFERENCES users(id),
    resolution_notes TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    resolved_at TIMESTAMP
);
```
